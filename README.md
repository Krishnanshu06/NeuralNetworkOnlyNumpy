# Neural Networks From Scratch (NumPy)

This project implements neural networks **from scratch using only NumPy**, without relying on machine learning frameworks like TensorFlow or PyTorch.

The goal of this project is to develop a **deep understanding of the mathematical and computational foundations of neural networks**, including forward propagation, backpropagation, and optimization.

---

## Motivation

Most machine learning libraries abstract away the internal workings of neural networks.  
This project recreates those mechanisms manually to understand:

- how neural networks compute predictions
- how gradients are calculated
- how weights are updated during training
- how optimization works in practice

---

## Features

- Fully implemented **feedforward neural network**
- **Forward propagation** implementation
- **Backpropagation algorithm** for gradient computation
- **Gradient descent optimization**
- Custom **loss functions**
- Implemented Multiple Sctivation Functions
- Support for multiple layers
- Training loop built entirely using **NumPy**
- Implemented regularization techniques
- Support for mini-batch training
- Implemented additional optimizers (Adam, RMSProp)

---

## Concepts Implemented

- Neural Networks

- Backpropagation

- Weight initialization

---

## Tech Stack

- **Python**
- **NumPy**

No deep learning frameworks were used.

---

## Example Workflow

1. Initialize network weights
2. Perform forward propagation
3. Compute loss
4. Perform backpropagation
5. Update weights using gradient descent
6. Repeat for multiple training iterations
